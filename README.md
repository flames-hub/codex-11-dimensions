# Codex 11 Dimensions

An interactive 1D–11D hypercube projection visualizer with bilingual English/Japanese explanations, dimension-specific observatory humor, and responsive controls.

## Live demo

https://flames-hub.github.io/codex-11-dimensions/

## Features

- Explore hypercube projections from 1D through 11D.
- Watch one true extruded Codex surface mesh collapse into 1D, flatten in 2D, rotate in 3D, and lift through compactified coordinates to an 11D brane.
- See dimension-specific signatures including temporal echoes, compactification loops, string harmonics, and the actual 240 roots of E₈.
- Let the guided specimen camera briefly face each new-dimensional form toward you before returning control.
- Slice 4D–11D forms with a live cross-section scanner along the newest extra axis.
- Trigger distinctive transformation events in 4D, 6D, 8D, 9D, 10D, and 11D.
- Turn the Codex specimen on or off; it starts ON on every visit.
- Choose a seamless fifteen-second smooth sweep or a thirty-second held sweep that pauses on each dimension and at 11D, then record it locally as MP4 or WebM with no upload.
- Rotate, zoom, pause, reset, and boost dense projection lines.
- Switch between English and Japanese.
- Read dimension-specific explanations, resident quotes, and observatory logs.
- Run as one self-contained HTML file with no external runtime dependencies.

## Run locally

Open `index.html` in a modern browser.
